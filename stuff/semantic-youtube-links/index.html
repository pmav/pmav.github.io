Semantic Youtube Links

Semantic Youtube Links is a Greasemonkey script that add semantic value to Youtube links.

This script append the title of the video to the URL. Example:

Before:
http://www.youtube.com/watch?v=AVbzsRxB5aU

After:
http://www.youtube.com/watch?v=AVbzsRxB5aU#universal-declaration-of-human-rights-in-passports

How to use it

Using Firefox:

  1. Install Greasemonkey.
  2. Install the script: semantic-youtube-links.user.js.
Using Google Chrome:
  1. Install the script: semantic-youtube-links.user.js.
