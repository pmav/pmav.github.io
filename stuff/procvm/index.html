procvm

procvm is a linux shell script that list VM RSS and VM Size (from /proc/<pid>), PID and comand for each running process.

Usage

Note: root access is needed. Example: 
chmod +x procvm.sh
sudo ./procvm.sh

PID     VmRSS   VmSize      Name (Path)
