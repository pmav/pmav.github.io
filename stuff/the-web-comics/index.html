About

The Web Comics is a RSS feed that agregates the follwing feeds:

  • Space Avalanche (feed)
  • Dilbert (feed)
  • Moronail (feed)
  • Cyanide and Happiness (feed)
  • Buttersafe (feed)
  • xkcd (feed)
  • LOL Cats (feed)
  • Troll Cats (feed)
  • Photoshop Disasters (feed)
  • The Dog House Diaries (feed)
  • PhD Comics (feed)
  • Reddit Webcomics (feed)
  • 9gag (feed)

You can get it here: The Web Comics RSS Feed. We are looking for new feeds, send your suggestions to @thewebcomics.

Yahoo! Pipes

We are using Yahoo! Pipes for the joining process, check out the source here: http://pipes.yahoo.com/pmav/thewebcomics.

Twitter

This feed is also consumed by a Twitter account: @thewebcomics. We use a in-house solution for this type of publishing.
